# README 設計

## 目的

READMEは簡潔型とする。変換仕様や検証項目を重複して詳述せず、概要と最小限の操作手順を示したうえで既存ドキュメントへ案内する。

## 明記する重要事項

- ファイルの変換はブラウザ内で完結し、外部APIへ送信しない。
- 複数のHPGL入力を、ファイルごとのレイヤーを持つ1つのDXFへ変換する。
- 対応拡張子は `.hpgl`、`.hpg`、`.plt`、`.h01`〜`.h99` とする。
- HPGLのペン番号をDXF図形のACI色として出力する。
- 固定スケールは40 HPGL単位を1 mmとする。
- Cloudflare Pagesはビルドコマンド `npm run build`、出力先 `dist` とする。
